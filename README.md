# Automação de Testes - BookStore API (Karate)

Este projeto consiste na automação de testes da API **BookStore** utilizando o framework **Karate DSL**.

## Pré-requisitos

Para executar o projeto, você precisa ter instalado e configurado corretamente em sua máquina:

* **Java JDK**
* **Maven**
* **IDE**

## Configuração da IDE

Recomenda-se instalar as seguintes extensões para melhor visualização dos arquivos `.feature`:

* **Cucumber (Gherkin) Full Support**
* **Cucumber for Java**

## Como Rodar

Para executar todos os testes, abra o terminal na raiz do projeto e execute o comando:

* mvn test

## Visualizando o Relatório
Ao final da execução, o terminal exibirá o local onde o relatório foi gerado. Basta copiar o caminho da URL do arquivo HTML exibido no terminal e colar no seu navegador para visualizar o resultado dos testes.
